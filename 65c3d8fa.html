阶段回顾

Spring核心容器

  • 依赖注入 DI
    • 自动化配置
      • Component
      • Autowired
        • 构造器
        • Setter方法
        • required=true
      • ComponentScan
    • JavaConfig：Configuration、Bean
    • XML方式
    • 混合配置
      • JavaConfig的导入
        • @Import（配置类.class...)
        • @ImportResource（xml文件）
      • XML中的导入
        • <import resource="xml文件"/>
        • <bean class="配置类">

面向切面编程

切面和业务逻辑的解耦

术语

Advice、Pointcut（表达式语法）、Aspect

分别代表：切什么、在哪里切、怎样切

两种支持类型

  1. @AspectJ注解驱动的切面
  2. XML配置
  3. @EnableAspectJAutoProxy

JDBC

Web开发框架（MVC、控制器、视图渲染）

  1. Servlet

    处理客户端请求（根据请求的URL决定将请求路由到哪个controller）

    Filter 过滤器

    在路由之前过滤请求（可能有多个，构成一个过滤链）

  2. Handler mapping

    servlet通过这个来确定url与controller的映射

  3. Controller

    处理请求（可能会调用service层、dao层…）

  4. Model and logical view name

    service层、dao层…的返回的结果Model，再加上逻辑视图作为Controller的总的返回值给Servlet

  5. ViewResolver

    结合Model和返回的视图做一个视图解析，如果解析到了某个页面，就会做Model和视图的结合，也就是渲染

  6. View

    最终渲染的结果页面

  7. Response

    携带结果返回

DispatchServlet配置

  • Servlet3规范
  • 使用Java将DispatchServlet配置到Servlet容器中（本质上是因为继承了WebApplicationInitializer

获得请求的输入

  • 查询参数=>?max=123 =>@RequestParam("max")
  • 路径参数 =>/{name} =>@PathVariable("name")
  • 表单参数=>参数名与对象字段名相同

测试控制器

  • org.springframework.test.web.servlet.MockMvc

安全与权限

逻辑：定义一个过滤器Filter，DelegatingFilterProxy将请求委托给FilterChainProxy

继承AbstractWeb

@EnableWebMvcSecurity

数据持久化

业务与访问持久化数据解耦

  • 数据访问对象DAO或Repository

    1. 方便业务层的测试

    2. 方便DAO层的替换

四种配置数据源的方式

  • JNDI
  • 连接池（第三方）
  • JDBC驱动程序定义
  • 嵌入式（内存数据库如H2）

Spring的JDBC框架

  • JDBC模板所做的事：资源管理和异常处理

  • ORM

    对象关系映射（数据库与Java对象之间的某种映射，以实现直接存取Java对象）

  • Hibernate三类查询

  • JPA——Java Persistence API

    • 宗旨是为POJO提供持久化标准规范
    • JPQL

  • SpringDataJpa

    • 加注解@EnableJpaRepositories
    • 继承接口JpaRepository

MongoDB

  • 注解@EnableMongoRepositories(basePackages="DAO层")

  • 继承MongoRepository

    （与JpaRepository一样来自CRUBRepository）

Redis数据类型（键值对中的value）

  • String
  • Linked Lists
  • Hashes
  • Sets

RedisTemplate

  • 简单的值
  • List类型的值
  • 在Set上执行操作
  • 绑定到某个Key上
