butterfly头参数大全

写法 解释
title 【必需】页面标题
date 【必需】页面创建日期
type 【必需】标签、分类和友情链接三個页面需要配置
updated 【可选】页面更新日期
description 【可选】页面描述
keywords 【可选】页面关键字
comments 【可选】提示页面评论模块(默认 true)
top_img 【可选】页面顶部图片
mathjax 【可选】提示mathjax(当设置mathjax的per_page: false時，才需要配置，默认 false)
katex 【可选】提示katex(当设置katex的per_page: false時，才需要配置，默认 false)
aside 【可选】提示侧边栏 (默认 true)
aplayer 【可选】在需要的页面加载aplayer的js和css,请参考文章下面的音乐 配置
highlight_shrink 【可选】配置代码框是否展开(true/false)(默认为设置中highlight_shrink的配置)
