SpringBot+Vue前后端分离开发

发表于 2021-01-08 更新于 2021-01-23

顺着b站的一个视频（链接在这）照猫画虎地写了个只能在本地运行的小程序

  1. springboot的创建以及配置

    • 将application.properties改为application.yml并改为以下内容

    • config中是解决跨域问题的代码（我也看不懂），具体解决了Vue前端无法访问后端8181端口内容的问题

    • controller中是功能实现

    • entity中是数据库实体，具体要根据数据库的表中内容来写

    • repository中是仓库（接口）

    • 主方法，不需要更改

    • test是一些测试，可要可不要。target则是打包后的程序（可以部署）

  2. Vue前端编写

    • 先看左侧的plugins：axios用来连接后端，element则是饿了么的ui，（这两者都是一开始没有的，在安装后被自动导入了的）

      都是通过npm install XXX --save安装的

      ​ =>简写：npm i XXX -S （注意S大写）

    • App.vue即主窗口，因为Vue的特点就是所有页面都在一个窗体显示，而APP窗体则是最外层的框架（无法被覆盖，一直存在）

    • router的index.js中则是左侧菜单栏（在APP.vue中被遍历）的具体内容

    • views中的文件则是实现具体功能的前端页面：

      • template中是页面的内容编写
      • script中是页面各个item的功能绑定

  3. 最后，也是暂时没试验的，就是项目的部署，在书上看到的具体步骤是：

    1. 用npm run build将Vue前端打包，会得到一个index.html和dist文件夹
    2. 将index.html和dist文件夹放到springboot项目的static文件夹下，然后再将springboot项目打包，得到一个target文件夹
    3. 最后将target文件部署就可以了。
