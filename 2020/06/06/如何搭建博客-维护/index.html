搭建博客全过程

发表于2020-06-06|更新于2021-01-23
|阅读量:|评论数:

适合小白的基础教程

  1. 下载node.js：直接百度然后安装
  2. win+r打开 cmd
    1)输入node -v检查node版本
    2)输入npm -v检查npm版本
  3. 输入npm install -g cnpm --registry=https://registry.npm.taobao.org然后等待安装完成
  4. 输入cnpm install -g hexo-cli等待安装完成
  5. 输入hexo -v你就会发现hexo被安装完成了，博客也就搭建完成了！

提示

默认目录：C:\User\你的电脑
可以通过输入d:来进入D盘
输入cd \XXX来进入XXX文件夹（如果XXX文件夹存在的话）

正式生成blog

  1. 输入mkdir blog你就会在当前目录下生成一个blog的文件夹，你所有写的博客的内容都会存储在这个文件夹内
  2. 输入cd blog/进入blog (可以输入pwd可以查看路径是否正确)
  3. 输入hexo init即利用hexo初始化一个博客
  4. 输入hexo s正式启动我们的博客 此时它会给我们生成一个本地的localhost:4000窗口，我们可以在浏览器中输入localhost:4000访问

写一篇文章

  1. 输入hexo n“我的第一篇博客文章”（可以输入dir查看当前目录的文件）
  2. 输入vim 我的第一篇博客文章.md然后进入这篇博客，给它增加些内容（我是用VScode打开md文件编辑的，更方便一点，不过记事本、Subline Text，Notepad++也都可以）
  3. 编辑完成后，输入hexo clean清除database，
    然后再输入hexo g生成一下，然后再输入hexo s启动，再用浏览器打开localhost:4000就可以本地访问了

至此，hexo博客的搭建已经完成了，下面我们需要的就是把文章部署到远端去，好让别人能看到

这里有一个免费的部署的方式就是部署到GitHub上去，通过GitHub的地址访问我们的博客

如何将我们的博客部署到GitHub上公开使用：

  1. 登录github 然后New repository
    注意！Repository name 一定要是 ：你的昵称.github.io
  2. 创建完成之后继续在cmd中：
    输入cnpm install --save hexo-deployer-git进行安装
  3. 在文件夹中找到_config.yml 文件，打开后在底部加上： 完成后保存退出
  4. cmd中输入hexo d然后登录GitHub就可以了
    （如果错误就试试加上一句： git config --global user.name '你的用户名'）
  5. 浏览器中打开网页：htts://你的昵称.github.io
  6. 大功告成！！！

更换主题：

这里只举一个简单地例子：

  1. cmd输入git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
    这样就把yilia的主题复制到themes/yilia文件夹中了
  2. 打开_config.yml 把theme的值更改成yilia
  3. 输入hexo clean
  4. 输入hexo g
  5. 输入hexo s
  6. 输入hexo d

搭建完成后的基本维护：

hexo n "新的文章"、 hexo clean、 hexo g、 hexo s、 hexo d按顺序来就行
注：有时候hexo d的时候弹出ERROR Deployer not found: git，就cnpm install --save hexo-deployer-git

更改博客目录
