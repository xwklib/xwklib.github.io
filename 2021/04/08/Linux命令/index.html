学习过的一些Linux命令

基本命令

tab可以补全目录或者文件两下tab可以查看提示（或帮助）

  1. pwd 查看当前根目录

  2. ls

    • ls /查看根目录下的文件或目录
    • ls -l 查看当前目录下的文件或目录详细信息
    • ls -a 查看当前目录下的所有文件或目录
    • ls –help 查看帮助

  3. cd /指定目录 切换目录

    • mkdir 创建文件夹
    • mkdir -p 创建文件夹（即使不存在）

  4. touch tmp.txt 创建文件

  5. rm

    • rm -r 删除目录
    • rm -rf 强制删除目录
    • rm 删除文件
    • rm -f 强制删除文件

  6. cp tmp.txt test1/test0

    将文件tmp.txt移到当前目录下的test1/tset0中

  7. mv tmp.txt /mnt/xmp2.txt

    将文件tmp.txt移动到与当前目录同级的mnt文件夹下并重命名为xmp2.txt

  8. ln

    • ln text1.txt /mnt/text2.txt 创建text1.txt的硬链接到/mnt下，并重命名为text2.txt

    • ln -s text1.txt /mnt/text3.txt 创建text1.txt的软链接到/mnt，并重命名为text3.txt

  9. cat

    • cat test.txt 查看test的数据

      • cat -n test2.txt > test2_1.txt

        test2_1.txt 中每一行都会加上行号

      • cat -b test2.txt > test2_1.txt

        test2_1.txt 中每一行（除了空行）都会加上行号

      cat test1.txt test2.txt > test3.txt则是将前两者合并成后者

      • cat > test4.txt << EOF 是创建test4然后进行文本输入，当输入EOF时终止
      • cat >> test4.txt << EOF 则是对已经存在的进行上述操作

  10. head

  • head test1.txt
  • head -n 5 test1.txt
  • head -n -20 test1.txt
  • head -c 10 test1.txt

查找命令

  1. fin /etc -name passwd 使用find查找、etc目录下的passwd文件
  2. which pwd 使用which查看pwd命令的路径
  3. whereis bash 查看bash的位置

压缩文件

  1. zip -r -q -o passwd.zip dwssap将dwssap文件压缩成passwd.zip

  2. rm -f dwssap删除文件

    unzip passwd.zip 解压文件

  3. tar -cf tar.tar * 将所有文件打包成tar1.tar

  4. tar -xvf tar1.tar -C /mnt/

文本编辑器

  1. 打开：vi file1.txt （或vim file1.txt）

  2. 按a进行编写（i则是插入），esc退出插入模式，按:进入命令模式，:q表示退出，:wq表示保存并退出，:wq!表示强制保存退出
