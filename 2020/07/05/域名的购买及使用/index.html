域名的购买及使用

发表于2020-07-05|更新于2021-01-23
|阅读量:|评论数:

很多网站都提供域名的购买，这里提供了腾讯云（也就是这篇文章啦）和godaddy（b站视频）两个，取其一就好。本文仅供借鉴

购买腾讯云域名（或其他）

  1. 进入 腾讯云 然后登陆（或者注册）

  2. 在腾讯云主页找到“域名注册”，找到自己喜欢的域名然后购买

  3. 购买完成之后，点击右上角的“控制台”，然后进入“域名管理”

  4. 查看域名的“服务状态”，可能需要实名认证，如果需要就按照腾讯云官方给的提示来就好，不需要的话请忽略这步

  5. 点击“操作”下面的“解析”，默认情况下是有两个记录的，不用管它

  6. 添加如下三条记录：

    主机记录 纪录类型 线路类型 记录值
    @ A 默认 192.30.252.153
    @ A 默认 192.30.252.154
    www CNAME 默认 你的github昵称.github.io(也就是你原本的博客地址)

    解释一下，192.30.252.153和192.30.252.154是github官方给出的ip地址（别人说的），但是我没法访问，很奇怪。

  7. 添加完之后，这里有两种操作方法

    • 直接在github的仓库里操作

    • 在本地cmd中操作
      第一种方法简单，但是如果要继续维护的话，还是避免不了第二种，所以我就直接说第二种了。

      • 在blog\source文件夹中生成一个CNAME文件（注意无后缀名）

      • 用记事本打开该文件然后添加你买的域名地址，保存并关闭

      • cmd中基本操作：hexo clean && hexo g && hexo d

    然后应该就可以了

附带一个不错的b站的教程，这里他用的是godaddy和dnspod，我就直接用了腾讯云，方便一点
