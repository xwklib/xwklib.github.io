Linux安装docker，用docker安装MySQL

发表于 2021-04-09 阅读次数： Valine：

参考文章：Linux 安装docker，以及docker的基本操作、Linux下 docker安装mysql 8 并配置远程连接

关于docker的作用，可以自己去百度或者CSDN上搜索相关文章查看，这里主要记录一些过程以及问题。

安装docker

Docker要求运行在Centos 7上，要求系统为64位，系统内核版本3.10以上

  1. uname -an查看当前系统版本

  2. yum -y install docker 下载安装docker

  3. service docker start2启动docker服务

  4. docker version 检查docker是否安装成功

当看到下图的信息，就是表示本机docker已经安装成功，很简单

镜像操作

搜索镜像

docker images II 查看下本机的是否已经有了镜像（默认是没有的）

docker search XXX搜索XXX的所有镜像，还能指定具体的版本进行下载，比如：docker search Ubuntu：1.2.5.4

下载镜像

docker pull docker.io/XXX 进行下载

下载到本地的镜像要比docker Hub上搜索出来的要大，因为下载过程中自动解压了，在查看镜像列表就有了刚才下载的镜像

列表包含了仓库名，版本标签，镜像ID，创建时间以及所占用的空间

删除镜像

docker rmi id号

rmi $\rightarrow$ remove id

剩下的还有很多，详细可以参看文章Linux 安装docker，以及docker的基本操作

想要进一步了解请自行百度

安装MySQL

  1. 下载MySQL镜像

    docker pull mysql

  2. 启动MySQL镜像

    docker run --name dockermysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=mysqlpassword -d mysql

    • –name 是镜像的别名
    • -p 将3306映射到3306（docker是个虚拟机有自己的端口）
    • -e MYSQL_ROOT_PASSWORD=mysqlpassword 设置mysql 服务器密码（后面需要，一定记住）
    • -d 后端启动
    • 启动的镜像名称（可以用id代替）

  3. 查询启动的镜像

    docker ps

  4. 进入容器

    docker exec -it dockermysql bash

    dockermysql是镜像的名字 ，可以用id替代

  5. 登录MySQL

    mysql -u root -p

    然后输入上面设置的密码

  6. 设置远程访问

    use mysql

    grant all privileges on *.* to 'root'@'%';
