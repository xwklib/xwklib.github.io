VScode，VS2019，Pycharm，Eclipse等快捷键合辑

总结一些常用工具的一些快捷键，也防止遗忘

  • 通用

    部分软件的快捷键可能与此部分不同，建议修改以便记忆，也有部分软件的快捷键不支持以下作用（或有其他作用），就放在后面的特殊部分了

    快捷键 解释（作用） 备注
    Ctrl+A 全选 all
    Ctrl+C 复制 copy
    Ctrl+F 查找 find
    Ctrl+K 复制当前行到下一行 K同“copy”
    注意部分软件 如vscode Ctrl+K与Ctrl+D是完全反过来的
    Ctrl+N 创建新项目 new
    Ctrl+O 打开项目 open
    Ctrl+P 打印（Linux系统为解锁） print
    Ctrl+R/H 替换 replace
    Ctrl+S 保存（Linux系统为锁定） save
    Ctrl+T 创建一个新开页面
    Ctrl+W 关闭当前页面
    Ctrl+X 剪切
    Ctrl+Z 撤销
    Ctrl+Y/Ctrl+Shift+Z 撤销“撤销”
    Ctrl+/ 注释
    Ctrl+F5 运行
    Alt+/ 内容提示/辅助
    Ctrl+Tab 切换到右邻页面
    Ctrl+Shift+Tab 切换到左邻页面

以下部分为软件特有快捷键，但也存在部分与其他软件相同的情况

  • Visual Studio 2019 （C++）

    快捷键 解释（作用） 备注
    Ctrl + K, Ctrl + D 代码格式化（即对齐）
    Ctrl + M, Ctrl + M/O 代码折叠/打开

    还有我自己习惯的，并且更改过了的一些快捷键，比如F5运行，F7生成什么的就不说了

  • Pycharm

    快捷键 解释（作用） 备注
    Shift+Enter 在当前行的下一行创建空白行
    Ctrl+D 复制当前行(或选中内容)到下一行(段) Duplicate Line or Selection
    Ctrl+Y 删除当前行

  • Eclipse

    快捷键 解释（作用） 备注
    Ctrl+Shift+O 组织类的import导入（既有Ctrl+Shift+M的作用（添加类的import导入），又可以帮你去除没用的导入，很有用）
    Ctrl+O 快速outline视图
    Shift+Enter 在当前行的下一行创建空白行
    Ctrl+D 复制当前行(或选中内容)到下一行(段) Duplicate Line or Selection
    Ctrl+Y 删除当前行
    Ctrl+E 等同于一般的Ctrl+Tab 建议改回去
    Ctrl+L 转到行…
    Ctrl+Shift+/ 代码折叠 注意是小键盘的/
    Ctrl+Shift+* 代码展开 注意是小键盘的*
    Ctrl+M 最大化当前窗口 再按反之

  • VsCode

    快捷键 解释（作用） 备注
    Shift+Enter 在当前行的下一行创建空白行
    !,Tab html必备操作 注意是先!然后再Tab
    Ctrl+Shift+P 调出控制台
    Ctrl+D 复制当前行(或选中内容)到下一行(段) Duplicate Line or Selection
    Ctrl+Y 删除当前行

  • Windows

    快捷键 解释（作用） 备注
    F2 重命名
    win+r 运行
    Ctrl+win+O 打开软键盘 但是一般会用到软键盘的情况都是没有键盘的情况，我也不知道开发者怎么想的
    鼠标右击底部任务栏，点击“显示触摸键盘按钮” 打开软键盘 然后点击任务栏右侧出来的键盘图标就可以使用了

  • Linux

    快捷键 解释（作用） 备注
    Tab 内容补全
    双击Tab 内容提示

  • Chrome浏览器

    快捷键 解释（作用） 备注
    Ctrl+D 收藏当前页面 其实对任何浏览器都适用
  • 文章目录
  • 站点概览
